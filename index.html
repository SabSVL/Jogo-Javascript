Jogo de colorir

Como jogar?

Utilize as setas do teclado para andar pela tela colorindo e fazendo desenhos da forma que preferir, faça uso de sua imaginição!

Teclas

  • Z = Para/voltar a colorir
  • R = Apagar o bloco um por um
  • W = Apagar todos os blocos
  • A = Mudar cor para azul
  • S = Mudar cor para verde
  • D = Mudar cor para amarelo
  • F = Mudar cor para rosa
  • G = Mudar cor para marrom
  • H = Mudar cor para laranja
  • J = Mudar cor para preto
  • K = Mudar cor para cinza
  • L = Mudar cor para vermelho

Boa sorte e Divirta-se!
